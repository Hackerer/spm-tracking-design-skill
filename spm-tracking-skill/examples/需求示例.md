# 需求示例 - 商品详情页埋点设计

> 本示例用于演示SPM埋点自动化设计的输入和输出

---

## 一、需求文档

### 1.1 业务背景

「京东闪购」频道首页需要进行改版，新增「限时抢购」模块，需要设计埋点方案来监控用户行为和转化效果。

### 1.2 核心业务流程

```
用户进入闪购首页
    ↓
浏览限时抢购模块（查看商品列表）
    ↓
点击商品卡片（进入商详）
    ↓
选择商品规格
    ↓
点击「立即购买」或「加入购物车」
    ↓
进入结算页
```

### 1.3 涉及页面

| 页面 | 页面路径 | 说明 |
|------|---------|------|
| 闪购首页 | /flash-sale/index | 闪购频道首页 |
| 商品详情页 | /product/detail | 商品详情 |
| 购物车页 | /cart/index | 购物车 |
| 结算页 | /checkout/index | 订单确认 |

### 1.4 关键指标

- 限时抢购模块曝光UV/PV
- 模块内商品点击率（CTR）
- 点击到加购转化率
- 点击到下单转化率

## 三、埋点方案输出

### 3.1 页面SPM映射

| 页面 | SPM a级 | SPM b级 | 说明 |
|------|---------|---------|------|
| 闪购首页 | flash | index | 闪购首页 |
| 商品详情页 | flash | product_detail | 商品详情页 |
| 购物车页 | cart | index | 购物车页 |
| 结算页 | cart | checkout | 结算页 |

### 3.2 埋点清单

| 序号 | 事件ID | 事件名称 | SPM | 事件类型 | biz_type | 触发时机 |
|-----|--------|---------|-----|----------|----------|----------|
| 1 | EVT001 | 首页浏览 | flash.index | pv | - | 进入首页 |
| 2 | EVT002 | 限时抢购曝光 | flash.index.limit_flash | 曝光 | - | 模块漏出 |
| 3 | EVT003 | 商品曝光 | flash.index.limit_flash.feeds.{n} | 曝光 | card | 卡片漏出50% |
| 4 | EVT004 | 商品点击 | flash.index.limit_flash.feeds.{n} | 点击 | card | 点击商品卡片 |
| 5 | EVT005 | 商详浏览 | flash.product_detail | pv | - | 进入商详 |
| 6 | EVT006 | 轮播图曝光 | flash.product_detail.banner | 曝光 | - | 轮播图漏出 |
| 7 | EVT007 | 轮播图点击 | flash.product_detail.banner.{n} | 点击 | image | 点击轮播图 |
| 8 | EVT008 | 规格选择 | flash.product_detail.info.spec | 点击 | select | 选择规格 |
| 9 | EVT009 | 收藏点击 | flash.product_detail.fixed_bottom.favorite | 点击 | favorite | 点击收藏 |
| 10 | EVT010 | 客服点击 | flash.product_detail.fixed_bottom.service | 点击 | service | 点击客服 |
| 11 | EVT011 | 加入购物车 | flash.product_detail.fixed_bottom.add_cart | 点击 | add_cart | 点击加购 |
| 12 | EVT012 | 立即购买 | flash.product_detail.fixed_bottom.buy_btn | 点击 | buy | 点击购买 |

### 3.3 参数定义

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| product_id | string | 是 | 商品ID |
| sku_id | string | 是 | SKU ID |
| price | number | 是 | 当前价格(分) |
| original_price | number | 否 | 原价(分) |
| quantity | number | 否 | 数量 |
| spec | string | 否 | 规格信息 |
| carousel_index | number | 否 | 轮播图索引(1,2,3) |
| favorite_status | boolean | 否 | 收藏状态 |

## 四、代码示例

### 4.1 商品曝光
```javascript
// SPM: flash.product_detail.feeds.{n}
// 事件类型: 曝光
SPM.trackExposure('flash.product_detail.feeds.1', {
  spm: 'flash.product_detail.feeds.1',
  biz_type: 'card',
  product_id: '123456',
  sku_id: 'sku_001',
  price: 9900,
  original_price: 19900
});
```

### 4.2 商品点击
```javascript
// SPM: flash.product_detail.feeds.{n}
// 事件类型: 点击
SPM.trackClick('flash.product_detail.feeds.1', {
  spm: 'flash.product_detail.feeds.1',
  biz_type: 'card',
  product_id: '123456',
  sku_id: 'sku_001',
  price: 9900
});
```

### 4.3 加入购物车
```javascript
// SPM: flash.product_detail.fixed_bottom.add_cart
// 事件类型: 点击
SPM.trackClick('flash.product_detail.fixed_bottom.add_cart', {
  spm: 'flash.product_detail.fixed_bottom.add_cart',
  biz_type: 'add_cart',
  product_id: '123456',
  sku_id: 'sku_001',
  price: 9900,
  quantity: 1
});
```

### 4.4 立即购买
```javascript
// SPM: flash.product_detail.fixed_bottom.buy_btn
// 事件类型: 点击
SPM.trackClick('flash.product_detail.fixed_bottom.buy_btn', {
  spm: 'flash.product_detail.fixed_bottom.buy_btn',
  biz_type: 'buy',
  product_id: '123456',
  sku_id: 'sku_001',
  price: 9900,
  quantity: 1
});
```
