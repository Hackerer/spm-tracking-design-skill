# SPM埋点方案文档

## 1. 需求概述

### 1.1 业务背景
疗愈空间是座舱内的情绪放松与恢复功能，提供"声音+视觉+座舱硬件联动"的疗愈体验。服务行程中的放松、冥想与身心调节需求。

### 1.2 涉及页面
| 页面b | 页面SPM | 说明 |
|------|---------|------|
| 放空 | healing.void | 放空模式（轻沉浸列表页/全沉浸播放页） |
| 冥想 | healing.meditation | 冥想模式（轻沉浸列表页/全沉浸播放页） |
| 呼吸 | healing.breathing | 呼吸调节（轻沉浸列表页/全沉浸播放页） |
| 播放器内核 | healing.player_core | 播放内核：起播、中止、完播、心跳 |

### 1.3 模式说明
| 模式 | 定位 | 音频来源 |
|------|------|----------|
| 放空模式 | 自然放松白噪声 | 网易云音乐 |
| 冥想模式 | 引导式冥想课程 | 网易云音乐 |
| 呼吸调节 | 呼吸调节课程 | 网易云音乐 |

## 3. 参数定义

### 3.1 公共业务参数 (HMI业务公参)
| 参数名 | 类型 | 必填 | 说明 | 枚举值 |
|--------|------|------|------|--------|
| user_pin | string | 是 | 用户京东pin | - |
| navi_bar_status | string | 是 | 导航栏状态 | 沉浸式/非沉浸式 |
| vid | string | 是 | 车辆id | - |
| server_env | string | 是 | 服务端环境 | live/staging/test |
| build_type | string | 是 | 包环境类型 | 可切环境包/不可切环境包 |
| order_id | string | 是 | jkc订单id | - |
| click_type | string | 是 | 点击触发方式 | voice/touch/gesture |

### 3.2 事件特有参数
| 参数名 | 类型 | 说明 | 枚举值 |
|--------|------|------|--------|
| immersion_state | string | 沉浸状态 | light/full |
| content_id | string | 内容ID | - |
| content_name | string | 内容名称 | - |
| content_duration | number | 内容时长(秒) | - |
| play_status | string | 播放状态 | playing/paused |
| play_progress | number | 播放进度(秒) | - |
| linkage_type | string | 座舱联动类型 | ambient_light/window/wind/reading_light |
| linkage_status | string | 联动状态 | on/off |

## 5. 座舱联动埋点

### 5.1 联动触发埋点

| 联动模块 | 触发时机 | 参数 |
|----------|----------|------|
| 氛围灯 | 进入播放态时打开，暂停/退出时恢复 | [公共参数] + linkage_type + linkage_status |
| 车窗 | 进入播放态时自动关闭，暂停/退出保持关闭 | [公共参数] + linkage_type + linkage_status |
| 风量 | 进入播放态时调至最低，暂停/退出保持低档位 | [公共参数] + linkage_type + linkage_status |
| 阅读灯 | 进入播放态时关闭，暂停/退出时恢复 | [公共参数] + linkage_type + linkage_status |

**说明：**
- 任一联动触发后，若被用户主动调整，则中断对应模块的联动，本次行程内不再次联动
